# 站点的展示

> *水库* *水文站*

- 默认先展示所有的站点（sttp为RR和ZZ）以及绑定tooltip
- 图标需要对应

- 每个站点需要有tooltip，默认重要站点的tooltip需要一直显示
- **水位/流量需要从接口中获取，数据拿到之后，通过stcd关联到相应的marker展示水位和流量**
- *水库* *水文站*图层的切换
- 点击列表点位到点击的站点

```
 图层组 dtx:dtxgroup
/** sttp
 * 水库
 *   RR
 *   水位/出库流量
 *  水文站
 *   ZZ
 *   水位/流量
 */
```
